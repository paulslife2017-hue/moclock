# MOCLOCK - 강남 프리미엄 헤드스파 & 두피 케어 웹사이트

## 📋 프로젝트 개요

**모클락(Moclock)**은 강남구 소재 프리미엄 두피 케어 및 헤드스파 서비스를 제공하는 브랜드의 공식 다국어 웹사이트입니다.

### 🏢 실제 업체 정보
- **위치**: 서울시 강남구 강남대로 520, 할리스커피 건물 2층
- **Google 평점**: ⭐ 4.8 / 5.0 (1,096+ 리뷰)
- **Naver 평점**: ⭐ 4.9 / 5.0 (850+ 리뷰)
- **특징**: 외국인 친화적 서비스, 영어 가능 직원

### 브랜드 의미
- **Moclock** = '모(毛)' + 'clock(시간)'의 합성어
- 탈모 관리의 **골든 타임**의 중요성을 강조
- 혁신적이고 차별화된 맞춤형 두피 케어 시스템 제공
